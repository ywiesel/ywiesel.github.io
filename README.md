# Yvette Wieseler Portfolio

This repository contains my personal portfolio website, built as a static site with HTML and CSS and hosted with GitHub Pages.

## Live Site

https://ywiesel.github.io

## Repository Structure

```text
.
├── index.html                  # Homepage
├── about.html                  # About and technical skills
├── projects.html               # Project descriptions and links
├── assets/
│   ├── css/
│   │   └── styles.css          # Shared site styling
│   ├── images/                 # Screenshots and image assets
│   └── presentations/          # Project presentation PDFs
└── docs/                       # Repository notes and maintenance docs
```

## Updating the Site

Keep public pages at the repository root so GitHub Pages can serve them directly. Place supporting files in `assets/` and use relative paths from each HTML page.
